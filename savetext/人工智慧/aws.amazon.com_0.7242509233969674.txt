人工智能 (AI) 是计算机科学的一个领域，致力于解决与人类智能相关的常见认知问题，例如学习、解决问题和模式辨识。人工智能 (通常简称为 "AI") 呈现出机器人或未来世界的景像，也就是说，AI 不再是科幻小说中虚构的机器人，而真正成为现代高端计算机科学中的现实。Pedro Domingos 教授是这个领域声誉卓着的研究人员，提出机器学习的「五大流派」，包括：符号主义 (Symbolists)，源自于逻辑和哲学；联结主义 (Connectionists)，神经科学的分支；进化主义 (Evolutionaries)，与进化生物学相关；贝叶斯派 (Bayesians)，采用统计学和概率方法；以及模拟主义 (Analogizers)，源自于心理学。最近由于统计运算效率的提高，所以贝叶斯派的学者成功地让「机器学习」在很多领域更向前迈进。同样地，网络运算方面的进展也让联结主义流派的学者进一步发展「深度学习」这个分支。机器学习 (ML) 和深度学习 (DL) 这两个计算机科学领域都是从人工智能这个学科衍生出来的。这些技术大体上分为「监督」和「未监督」两种学习技巧，「监督」使用包含预期结果的训练数据，「未监督」则使用不包含预期结果的训练数据。提供 AI 越多数据，它就会变得「越聪明」而且学习更快，企业每天产生这些数据让机器学习和深度学习解决方案更加完备，包括从数据仓库 (如 Amazon Redshift) 收集或截取的数据、透过 Mechanical Turk 的「群众」力量所产生的真实数据，或是透过 Kinesis Streams 动态探勘得到的数据等等。此外，由于 IoT 的出现，感应器技术让需要分析的数据量大幅暴增，这些数据报含之前几乎不曾接触过的来源、地点、对象及事件数据。

机器学习是多种贝叶斯派技巧的通称，用于模式辨识和学习。机器学习的核心是算法的集合，可根据记录的数据学习和进行预测、优化不确定情况下的特定公用程序函数、从数据截取隐藏的结构，以及将数据分类成精确的描述。如果明确的编程太过于死板或不实际，通常会使用机器学习。相较于软件开发人员为了根据特定输入来产生代码特定输出所开发的一般电脑代码相比，机器学习使用数据产生统计代码 (ML 模型)，它会根据从之前输入范例 (在监督学习技巧中也包括输出范例) 辨识的模式输出「正确的结果」。ML 模型的正确性主要取决于过去数据的质和量。有了正确的数据，ML 模型可以利用数十亿个范例来分析高维度问题，以找出可根据特定输入预测结果的优化函数。ML 模型通常可根据统计方法提出可靠的预测及整体性能。如果您要使用 ML 模型或任何个别预测，这类评估分数对于决策而言很重要。

Amazon.com 将许多业务创建在以机器学习为基础的系统上。如果没有 ML，Amazon.com 就无法扩展业务、提升客户体验和选择，以及优化物流速度和品质。Amazon.com 创立 AWS 是为了让其他企业也能享有相同的 IT 基础架构，并充分利用其灵活性和成本效益，我们现在仍持续致力于让 ML 技术普及到各个企业。Amazon.com 开发团队的结构及对运用 ML 来解决困难的实际商业问题的专注，促使 Amazon.com 与 AWS 开发易于使用且功能强大的 ML 工具及服务。这些工具就像其他 IT 服务一样，会先在 Amazon.com 内部和关键任务环境中进行测试，然后才以 AWS 服务的形式公开供各个企业使用。

机器学习通常用来根据过往的数据来预测未来的结果。例如，组织会使用机器学习根据特定人口统计学来预测他们的产品在未来会计年度的销售量；或是预测具备哪些特性的客户对品牌不满意或最忠实的可能性最高。这类预测有助于做出更好的商业决策、提供更贴近个人的用户体验，以及具备降低消费者忠诚度成本的潜力。ML 可补足商业智能 (BI) (主要在于报告过去的商业数据) 不足的地方，可根据以往的趋势和交易来预测未来的结果。在企业中成功实作 ML 有几个步骤。首先，找出真正的问题；识别明确后可让企业受益的预测。接着，必须根据以往的业务指针 (交易、销售、损耗等等) 收集数据。数据汇总之后，即可依据这些数据创建 ML 模型。运行 ML 模型并将模型的预测输出套用回商业系统，以做出更明智的决策。

深度学习是机器学习的分支，它涉及分层算法，目的是更深入地了解数据。算法不再局限于创建一组可解释的关系做为更基本的回归。反之，深度学习倚赖非线性算法的这些分层，以创建依据一系列因子交互的分布式表示法。为深度学习提供大量的训练数据之后，就能开始识别这些元素之间的关系。这些可能是图形、色彩、字词等之间的关系。之后，就能使用系统来创建预测。在机器学习和人工智能中，系统的深度学习主体能够比人类实际在软件编写的代码识别更多关系，或者甚至识别人类无法察觉的关系。经过足够的训练之后，就能让算法网络开始对非常复杂的数据进行预测或解译。
